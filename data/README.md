# Data

This folder contains the cleaned datasets used across the ECOWAS Language & 
Digital Identity Inclusion dashboard.

Files:
- ECOWAS_Master_Data.xlsx — country-level language, literacy, internet access, 
  and gender inclusion indicators
- Portal_Audit.xlsx — government digital ID portal accessibility audit

All datasets are cleaned and standardized by the Data Strategy team before 
being used for analysis or visualization. Raw/uncleaned versions are not 
stored here
